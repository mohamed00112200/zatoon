## 📌 Overview

Zatoon (Elemanolive) is a professional landing page / business website designed for an olive production and pickles company.

The website includes:

✅ Company introduction  
✅ Product offers  
✅ Product menu  
✅ Image gallery  
✅ Contact information  
✅ Social media links  
✅ Responsive design for mobile & desktop  

Built using only HTML + CSS without frameworks.
